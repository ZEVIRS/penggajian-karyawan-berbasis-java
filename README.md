# Java GUI Payroll System

Aplikasi Penggajian Karyawan berbasis Java GUI yang dibuat untuk membantu proses pengelolaan data karyawan dan perhitungan gaji secara terstruktur dan otomatis.

Project ini dikembangkan sebagai bagian dari tugas mata kuliah **Pemrograman Java / Pemrograman Berorientasi Objek**.
